# 19-react-portfolio-conversion

This project demonstrates how to transform a static portfolio website into a component-based React application. The application uses React functional components, props, and hooks for better code organization and scalability.



 Features

 React component architecture
 Reusable components
 Props implementation
 State management using `useState`
 Responsive design
 Dynamic project rendering
 Contact form handling
 Clean folder structure



 Technologies Used

 React JS
 JavaScript (ES6)
 HTML5
 CSS3
